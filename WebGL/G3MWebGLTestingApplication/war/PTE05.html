Glob3 Mobile API - Ejemplo capas WMS

Ejemplo de mostrar capas WMS incluidas en el PTE05 del Cabildo de Gran Canaria

Esta demo muestra cómo se pueden incluir todas las capas WMS que se quieran sobre el globo, por medio de transparencias. Actualmente se están mostrando las siguientes capas:

  • Capa Ortoexpress como capa base, sobre la que se pintará el resto

  • Capa de predominancia de la Movilidad en viario (líneas en violeta)

  • Catálogo de Áreas de Sensibilidad Ecológica (zonas verdes semitransparentes)

  • Ámbitos de Predominancia Agrícola-urbano (líneas amarillas)


Para manejar el globo se usan estos controles:

  • Botón izquierdo apretado del ratón para trasladar el terreno

  • Boton izquierdo del ratón aprentado + tecla Shift para abatir la cámara (moviendo ratón en vertical)

  • Boton izquierdo del ratón aprentado + tecla Alt para orbitar alrededor de nuestra posición

  • Rueda del ratón para hacer zoom adelante y atrás
